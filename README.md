# Rokhdad / رخداد

Domain: `rokhdad.top`

Rokhdad is a separate server-only, GitHub-first project for Persian event aggregation, discovery, registration, ticketing, and organizer tooling.

This repository is intentionally in planning mode. Per project rules, implementation must start only after choosing a specific task ID from `docs/TASK_BOARD.md`.

## Project Rules

- Do not implement multiple unrelated tasks in one pass.
- Local development must not require Docker, PHP, Node.js, Python, MariaDB, MongoDB, Redis, or Nginx.
- Runtime services must run on the Ubuntu server using Docker Compose.
- Code must be pushed to GitHub first.
- The server must pull from GitHub and run Docker Compose.
- Production domain is `rokhdad.top`.

## Current Status

- Planning: in progress.
- Product code: not started.
- Deployment: blocked until GitHub repo and server access are confirmed.
